⚠️ IMPORTANT WARNING BEFORE RUNNING! ⚠️
This pack will not run on its own. For the characters to move and play audio on your PC, you must have Python and the Pygame library installed.

STEP-BY-STEP INSTALLATION GUIDE:
1. Download and install the latest Python from the official website (python.org).
   *CRITICAL:* During installation, make sure to check the "Add python.exe to PATH" box.
2. Open Windows Command Prompt (CMD), paste the following command, and hit Enter:
   pip install pygame

ABOUT THIS PROJECT
-------------------------------------------------------------------------------
Bring the epic world of Animation vs. Animator directly to your Windows desktop! This is a 100% custom, fan-made interactive stickman pack built completely from scratch using pure Python and Tkinter. 

This is NOT a Shimeji clone – every character runs on its own advanced AI, features unique combat modes, and comes with full audio integration!

KEY FEATURES & CHARACTER GUIDES
-------------------------------------------------------------------------------

1. THE SECOND COMING (TSC)
* Appearance:         - Orange, hollow-headed stickman with thin lines.
* Przycisk [ X ]      - Red square in the top-right corner of the screen. 
                        Clicking with the mouse instantly closes the character.
* Key [ F ]           - Awakening Mode: Triggers the bright green, tearing aura 
                        of lightning and electrical discharges from AvA 8.
* Key           - Infinite Laser: Works ONLY in Awakening Mode. TSC freezes 
  (Hold)                in place and shoots a green beam from his eyes that cuts 
                        the monitor to the very edges without a range limit (5000px). 
                        Generates loud audio from sounds/laser.wav. Any other 
                        character that TOUCHES the beam line is instantly wiped 
                        out and removed from the screen!
* Key           - Autonomous AI Flight: Works ONLY in Awakening Mode. TSC 
                        evades gravity and COMPLETELY AUTONOMOUSLY drifts, hovers, 
                        and orbits around the screen (ignores the mouse).
* Key           - Viewer / Fan Mode: TSC shrinks to 1.1 scale, flies to the 
                        bottom-right corner, and physically SITS on his side, 
                        bending knees at a 90-degree angle, with both legs pointed 
                        perfectly to the LEFT, waving his hands.

2. RED

                        Minecraft. His autonomous AI gains massive running speed.
* Key [ Z ]           - Minecraft Cake Throw: Works in G mode. Red throws a square 
                        cake towards your mouse. Upon reaching the mouse point, 
                        the cake explodes and leaves a cream stain with cherries 
                        on the monitor for exactly 5 seconds.

3. GREEN
* Key [ Y ]           - Influenced Mode: Green walks forward staring into his 
                        smartphone. After colliding with a wall, he trips and falls.

4. BLUE
* Key [ O ]           - Speed Potion: Blue gets an enormous sprint speed boost 
                        and leaves bright blue trails of momentum behind him.

5. YELLOW
* Key [ P ]           - Scripter mode: Yellow walks with an open laptop showing 
                        flashing green code, stopping randomly to ponder.

6. THE CHOSEN ONE (TCO)
* Key [ T ]           - Flight Mode: TCO evades gravity and flies autonomously, 
                        with fire blasting downwards from his hands.
* Key [ R ]           - Red Eye Laser: Shoots a powerful, unlimited red beam 
  (Hold)                across the entire monitor towards the mouse cursor.
* Key [ L ]           - Fire Stream from Hands: Blasts powerful fire towards 
  (Hold)                the mouse. TCO freezes in place (Locked during flight).

7. THE DARK LORD (TDL)
* Key [ X ]           - Blade Mode: Deploys huge black blades from his arms. Gains 
                        massive running and jumping speed, with sparks flying from hands.
* Key           - Fireball: Launches a red, burning ball from his palm 
                        directly at the mouse cursor position.
* Key [ V ]           - X-Slash: TDL executes a powerful slash and releases a giant

                        Virabots from the desktop.
* Key [ - ]           - Flight Mode: TDL rises into the air and flies autonomously 
                        in an aerodynamic, vertical position with arms down.

8. VICTIM / H4CK3R
* Key           - H4CK3R Mode: Victim triggers a pulsing neon aura,

* Key           - Hacker Flight: Works ONLY in mode 6. Victim evades gravity 
                        and flies autonomously across your screen.
* Key [ M ]           - Hacker Cage: Creates a large, neon 180x180 cage under the

* Key [ N ]           - Remove Cages: Instantly destroys all cages and prematurely 
                        frees your mouse cursor.
* Key [ B ]           - Mouse Hijack: For 3 seconds, the mouse cursor on the screen

9. KING ORANGE
* Appearance:         - Character size 1.7. By default, walks as a clean stickman.
* Key [ H ]           - King Mode (Transformation): Summons the power of the blocks!

GLOBAL INTERACTIONS (HACKING PLATFORMS)
-------------------------------------------------------------------------------
* Key [ S ]           - Hover your mouse over the top bar of any application (e.g., 
                        Roblox, Discord) or an icon and press S to create a stable 
                        floor for all walking stickmen.
* Key [ U ]           - Resets and clears all created S platforms from memory.
* FOCUS NOTE          - If a stickman does not respond to keys, click on him once 
                        with the Left Mouse Button (LMB) so Windows restores focus to him.
